# 问题背景
一个班40份作业，每份都需要判对错、写评语、分析薄弱点——这是老师们每天都在面对的高强度重复工作。本赛道邀请参赛者打造一款AI智能作业批改系统，支持多学科作业的自动批改，不仅能判断对错，更能输出个性化评语和知识点薄弱分析，真正做到"每个学生都被认真对待"

# 核心能力要求
1.多学科支持：至少覆盖2个学科（如数学计算题批改 + 语文/英语作文评改）
2.手写体识别：准确识别学生手写内容，包括公式、汉字、英文等
3.智能判分：不仅判断对错，还能识别部分正确的解题过程并给予过程分
4.个性化评语生成：针对每份作业生成有针对性的评语，而非千篇一律的模板
5.知识点薄弱分析：汇总学生的错误模式，定位薄弱知识点，输出学情分析报告

# 交付要求
1.可运行的多学科批改Demo
2.批改样例展示：输入作业图片 → 输出批改结果 + 评语 + 薄弱点分析
3.技术说明文档（OCR方案、知识建模、评语生成策略）
